# MSIX Asset Requirements

Place the following PNG files in this directory for Windows Store submission:

| File | Size | Description |
|------|------|-------------|
| `StoreLogo.png` | 50x50 | Store listing icon |
| `Square44x44Logo.png` | 44x44 | Taskbar / small tile |
| `Square150x150Logo.png` | 150x150 | Medium tile |
| `Wide310x150Logo.png` | 310x150 | Wide tile |
| `LargeTile.png` | 310x310 | Large tile (optional) |
| `SplashScreen.png` | 620x300 | Splash screen (optional) |

All images should have a transparent background with the PurpleSector logo.
The recommended brand color is `#7C3AED` (purple-600).
